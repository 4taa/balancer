# balancer

## Install && start
`npm use | npm install | npm start`

### Server for metrics
`http://${hostname}/metrics`

## Server for requests 
`http://${hostname}/api/*`

### API Doc

`/api/check` - is server available for requests

`/api/off` - turns server off

`/api/on` - turns server on

`/api/data` - return random data to user
